27 /// The code generation environment class aggregates a number of data
28 /// structures that are needed during the code generation phase of
29 /// sparsification. This environment simplifies passing around such
30 /// data during sparsification (rather than passing around all the
31 /// individual compoments where needed). Furthermore, it provides
32 /// convience methods that keep implementation details transparent
33 /// to sparsification while asserting on internal consistency.
34 class CodegenEnv {
35 public:
36  /// Constructs a code generation environment which can be
37  /// passed around during sparsification for bookkeeping
38  /// together with some consistency asserts.
39  CodegenEnv(linalg::GenericOp linop, SparsificationOptions opts,
40  unsigned numTensors, unsigned numLoops, unsigned numFilterLoops);
